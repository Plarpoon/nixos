This repository contains the configuration for my **daily driver** NixOS machine. I use this to track system changes and maintain a reproducible environment.

> [!CAUTION]
> This configuration is tailored to my specific hardware. If you are forking this, ensure you update the `hardware-configuration.nix` and the kernel paramters to match your own machine to avoid boot issues.
